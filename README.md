# Recycleware - Plataforma de Reciclaje de Hardware

Proyecto web para la gestión y donación de hardware reciclado (Recycleware).

## 🛠️ Tecnologías

### Frontend
* **HTML5 & CSS3**
* **Sass (SCSS):** Para estilos avanzados y gestión de variables.
* **Bootstrap 5:** Framework CSS base.
* **JavaScript:** Lógica de interfaz.

### Stack Futuro (Migración planificada)
* **Frontend:** React.js
* **Backend:** Java + Spring Boot
* **Base de Datos:** MySQL

## 📂 Estructura del Proyecto

El proyecto está dividido en dos directorios principales para facilitar la escalabilidad:

* `/frontend`: Contiene toda la interfaz de usuario.
* `/backend`: Contiene la lógica del servidor y APIs.

## 🚀 Cómo empezar (Fase HTML/Sass)

1.  Navega a la carpeta frontend:
    `cd frontend`
2.  Instala las dependencias (Bootstrap, Sass):
    `npm install`
3.  Compila el SASS:
    `npm run sass`
4.  Abre `index.html` en tu navegador.
